## Features

### Required Features
- [x] Use a logical component structure in React
- [x] Display at least five content creators on the homepage
- [x] Each creator shows their name, a link, and a description
- [x] API calls use async/await pattern
- [x] Clicking a creator goes to their details page
- [x] Each creator has their own unique URL
- [x] User can edit a creator
- [x] User can delete a creator
- [x] User can add a new creator
- [x] New creator appears in the list

### Stretch Features
- [x] Display creators as cards
- [ ] Use PicoCSS for styling
- [ ] Show images on creator cards

## Tech Stack
- React
- Vite
- Supabase
- React Router

## How to Run
1. Clone the repo
2. Run `npm install`
3. Add your Supabase URL and API key to `src/client.js`
4. Run `npm run dev`
